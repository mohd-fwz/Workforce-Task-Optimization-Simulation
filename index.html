Workforce Optimizer

NGO Emergency Response & Task Management Dashboard

⚠️ Setup Required
This application requires installation and setup. Please follow the instructions below.

Quick Start

Open terminal in this directory
Run: npm install
Run: npm run install:all
Run: npm run dev
Open: http://localhost:3000

What You'll Get

📊Live Data Structure Viz
🗺️Interactive Map
⚡Real-time Updates
🎯Smart Allocation
📈Priority Queue (Heap)
🔗Graph Pathfinding

Technologies

Backend: Node.js + Express + Socket.IO
Frontend: React + TypeScript + D3.js + Leaflet
Algorithms: Min Heap, Dijkstra, FIFO Queue, HashMap
